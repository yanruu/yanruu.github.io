ARIMA는 선형적인 자기상관 구조만 다룰 수 있다는 한계가 있었다. 이번 글부터는 비선형 패턴까지 학습할 수 있는 딥러닝 계열로 넘어간다. 그 출발점이 **RNN(Recurrent Neural Network, 순환 신경망)** 이다.

## 왜 일반 신경망으로는 시계열을 다루기 어려운가

일반적인 완전연결(Fully Connected) 신경망은 입력 하나하나를 독립적으로 처리한다. 하지만 시계열은 순서가 중요하다 — 어제 값이 오늘 값에 영향을 주고, 그 영향은 다시 내일로 이어진다. RNN은 이 "이전 정보를 기억해서 다음 계산에 넘겨주는" 구조를 신경망에 추가한 것이다.

## RNN의 핵심 — 은닉 상태(Hidden State)

RNN은 매 시점마다 입력 $x_t$와 이전 시점의 은닉 상태 $h_{t-1}$을 함께 받아서 새로운 은닉 상태 $h_t$를 만든다.

$$h_t = \tanh(W_h h_{t-1} + W_x x_t + b)$$
$$\hat{y}_t = W_y h_t + b_y$$

여기서 핵심은 $W_h$, $W_x$가 **모든 시점에서 동일하게 공유**된다는 점이다. 같은 가중치를 반복 사용하기 때문에 시퀀스 길이가 달라져도 같은 모델로 처리할 수 있다.

![RNN을 시간 축으로 펼친 모습](/assets/images/rnn_unrolled_diagram.png)
*같은 셀이 시간 순서대로 반복되며, 은닉 상태가 화살표를 따라 다음 시점으로 전달된다. 가중치 $W_h$는 모든 시점에서 동일하다 — 이것이 "순환(recurrent)"이라는 이름의 이유다.*

## PyTorch로 구현해보기

```python
import torch
import torch.nn as nn

class SimpleRNN(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super().__init__()
        self.rnn = nn.RNN(input_size, hidden_size, batch_first=True)
        self.fc = nn.Linear(hidden_size, output_size)

    def forward(self, x):
        out, hidden = self.rnn(x)        # out: (batch, seq_len, hidden_size)
        last_out = out[:, -1, :]          # 마지막 시점의 은닉 상태만 사용
        return self.fc(last_out)

model = SimpleRNN(input_size=1, hidden_size=16, output_size=1)
print(model)
```

`nn.RNN`은 시퀀스 전체를 한 번에 넣으면 내부적으로 시점마다 순환을 반복하며 모든 시점의 출력을 돌려준다. 예측에는 보통 마지막 시점의 은닉 상태만 사용한다.

## RNN의 한계 — 기울기 소실(Vanishing Gradient)

RNN은 역전파 시 시간을 거슬러 올라가며 같은 가중치 $W_h$를 반복해서 곱한다. $W_h$의 값이 1보다 작으면 먼 과거로 갈수록 기울기가 기하급수적으로 작아진다 — 이를 **기울기 소실 문제**라 한다.

```python
import numpy as np

steps = np.arange(1, 51)
grad_rnn = 0.85 ** steps   # 매 시점 0.85씩 곱해진다고 가정
```

![시퀀스 길이에 따른 기울기 크기 변화](/assets/images/vanishing_gradient.png)
*RNN(빨강)은 약 20스텝만 지나도 기울기가 거의 0에 가까워진다. 즉 20스텝 이전의 정보는 학습에 거의 반영되지 못한다. LSTM(파랑)은 게이트 구조 덕분에 훨씬 느리게 감소한다.*

이 그래프가 RNN의 근본적인 약점을 보여준다. "오래 전 정보를 기억해야 하는" 문제일수록 RNN은 불리하다 — 예를 들어 30일 전 패턴이 오늘 값에 영향을 준다면, 순수 RNN은 그 관계를 거의 학습하지 못한다.

## RNN vs 이전 모델들

| 항목 | ARIMA | RNN |
|------|-------|-----|
| 패턴 | 선형 | 비선형 가능 |
| 장기 의존성 | 차분/계절성 항으로 제한적 처리 | 이론상 가능하나 기울기 소실로 실제는 어려움 |
| 다변량 | SARIMAX로 확장 | 입력 차원만 늘리면 자연스럽게 확장 |
| 학습 난이도 | 낮음 | 높음 (데이터·튜닝 필요) |

기울기 소실 문제를 구조적으로 해결한 것이 다음 글에서 다룰 **LSTM**이다. 게이트(gate)라는 장치를 추가해서 "무엇을 기억하고 무엇을 잊을지"를 모델이 직접 학습하게 만든다.

다룰 내용:

- 하이브리드: AWS Storage Gateway (온프렘 ↔ 클라우드 캐시 연결)
- 라이프사이클 정책 — 자동으로 저렴한 클래스로 이동
- AWS Backup — 여러 서비스 백업 통합 관리

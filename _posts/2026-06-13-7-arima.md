지난 글에서 정상성과 ACF/PACF를 배웠다. 이번 글에서는 이를 실제로 활용하는 가장 기본적인 시계열 예측 모델, **ARIMA**를 다룬다. ARIMA가 무엇의 약자인지부터, 차분, 차수 결정, 실제 예측까지 순서대로 진행한다.

## ARIMA = AR + I + MA

ARIMA(p, d, q)는 세 요소의 조합이다.

- **AR(p)**: 자기회귀(AutoRegressive). 과거 p개 시점의 값으로 현재를 예측.
- **I(d)**: 차분(Integrated). 비정상 시계열을 d번 차분해서 정상으로 만든다.
- **MA(q)**: 이동평균(Moving Average). 과거 q개 시점의 오차(잔차)로 현재를 보정.

$$y_t = c + \sum_{i=1}^{p}\phi_i y_{t-i} + \sum_{j=1}^{q}\theta_j \epsilon_{t-j} + \epsilon_t$$

## 1단계 — 정상성 확인 및 차분

지난 글에서처럼 ADF 검정으로 비정상임을 확인했다면, 차분으로 정상화한다.

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.stattools import adfuller

np.random.seed(0)
t = np.arange(120)
data = 0.3 * t + np.random.normal(0, 3, 120)
ts = pd.Series(data, index=pd.date_range('2015-01', periods=120, freq='ME'))

print("차분 전 ADF p-value:", adfuller(ts)[1])

ts_diff = ts.diff().dropna()
print("1차 차분 후 ADF p-value:", adfuller(ts_diff)[1])
```

![차분 전후 비교](/assets/images/arima_differencing.png)
*위는 추세가 뚜렷한 원본 시계열(비정상), 아래는 1차 차분 후 — 추세가 사라지고 0을 중심으로 흔들리는 정상 시계열이 됐다.*

차분 한 번으로 p-value가 0.05 밑으로 떨어지면 d=1로 충분하다는 뜻이다. 대부분의 실제 데이터는 d=0, 1, 2 중 하나로 해결된다.

## 2단계 — ACF/PACF로 p, q 차수 정하기

차분된 시계열의 ACF/PACF를 보고 p, q의 후보를 정한다. PACF가 시차 1 이후 급격히 작아지면 AR(1), ACF가 시차 1 이후 급격히 작아지면 MA(1)을 의심한다.

```python
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
plot_acf(ts_diff, lags=20, ax=axes[0])
plot_pacf(ts_diff, lags=20, ax=axes[1], method='ywm')
plt.show()
```

## 3단계 — 모델 적합 및 예측

```python
from statsmodels.tsa.arima.model import ARIMA

model = ARIMA(ts, order=(1, 1, 1))
fit = model.fit()
print(fit.summary())

forecast = fit.get_forecast(steps=12)
mean_forecast = forecast.predicted_mean
conf_int = forecast.conf_int()
```

`order=(1,1,1)`은 AR(1) + 1차 차분 + MA(1)이라는 뜻이다. `fit.summary()`로 각 항의 계수와 p-value, AIC를 확인해 모델이 통계적으로 유의한지 점검한다.

![ARIMA 예측 결과](/assets/images/arima_forecast.png)
*실선이 과거 실제값, 주황색이 향후 12개월 예측값, 음영이 95% 신뢰구간이다. 예측이 진행될수록 신뢰구간이 넓어지는 것은 먼 미래일수록 불확실성이 커진다는 자연스러운 결과다.*

## 차수를 자동으로 찾기 — auto_arima

p, d, q를 직접 하나씩 시도하는 대신 `pmdarima`의 `auto_arima`로 AIC 기준 최적 차수를 탐색할 수 있다.

```python
from pmdarima import auto_arima

auto_model = auto_arima(ts, seasonal=False, trace=True,
                         stepwise=True, suppress_warnings=True)
print(auto_model.summary())
```

`trace=True`로 두면 여러 조합을 시도하는 과정과 각 AIC 값을 모두 출력해줘서, 왜 특정 차수가 선택됐는지 확인할 수 있다.

## ARIMA의 한계

ARIMA는 선형적인 자기상관 구조만 포착한다. 계절성이 강하면 SARIMA로 확장해야 하고, 변수 간 복잡한 비선형 관계나 다른 시계열의 영향을 함께 고려해야 한다면 ARIMA로는 한계가 있다. 이 한계가 이후 글에서 다룰 RNN/LSTM 계열 모델로 넘어가는 이유다.

| 항목 | ARIMA | 딥러닝(RNN/LSTM) |
|------|-------|------------------|
| 가정 | 선형적 자기상관 | 비선형 패턴도 학습 가능 |
| 필요 데이터량 | 적어도 동작 | 비교적 많아야 안정적 |
| 해석 가능성 | 높음 (계수 직접 해석) | 낮음 (블랙박스) |
| 다변량 확장 | 제한적 (SARIMAX 등) | 자연스러움 |

다음 글부터는 본격적으로 딥러닝 계열로 넘어간다. 시계열을 순차적으로 처리하는 신경망 구조인 RNN부터 시작한다.

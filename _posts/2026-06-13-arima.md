## 들어가며

[지난 글](../time-series-basics)에서 ACF/PACF로 AR, MA 모델의 차수를 결정하는 방법을 배웠다.
**ARIMA**는 AR, 차분(I), MA를 결합한 시계열 예측의 대표 모델이다.
딥러닝 이전까지 시계열 예측의 표준이었으며, 지금도 베이스라인으로 자주 사용된다.

---

## 1. AR 모델 (AutoRegressive)

**AR(p)**: 과거 $p$개의 값으로 현재를 예측

$$y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \cdots + \phi_p y_{t-p} + \epsilon_t$$

- $\phi$ : AR 계수
- $\epsilon_t$ : 백색 잡음(white noise)
- PACF에서 p 시차 이후 절단되면 AR(p) 적합

---

## 2. MA 모델 (Moving Average)

**MA(q)**: 과거 $q$개의 오차항으로 현재를 예측

$$y_t = c + \epsilon_t + \theta_1\epsilon_{t-1} + \theta_2\epsilon_{t-2} + \cdots + \theta_q\epsilon_{t-q}$$

- $\theta$ : MA 계수
- ACF에서 q 시차 이후 절단되면 MA(q) 적합

---

## 3. ARIMA(p, d, q)

**ARIMA = AR(p) + 차분(I, d) + MA(q)**

- $p$ : AR 차수 (PACF로 결정)
- $d$ : 차분 횟수 (정상화에 필요한 차분 수)
- $q$ : MA 차수 (ACF로 결정)

비정상 시계열을 $d$번 차분해서 정상화한 뒤 ARMA(p,q)를 적용한다.

---

## 4. ARIMA 적용 순서

**Step 1. 정상성 확인 → 차분**

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.stattools import adfuller
import matplotlib.font_manager as fm

font_candidates = [
    'Noto Sans CJK JP', 'Noto Sans CJK KR',
    'NanumGothic', 'Malgun Gothic', 'AppleGothic', 'DejaVu Sans'
]
available_fonts = {f.name for f in fm.fontManager.ttflist}
selected_font = next((f for f in font_candidates if f in available_fonts), 'DejaVu Sans')
plt.rcParams['font.family'] = selected_font
plt.rcParams['axes.unicode_minus'] = False

# 예시 데이터
np.random.seed(42)
t = np.arange(120)
data = 0.3 * t + np.random.normal(0, 3, 120)
ts = pd.Series(data, index=pd.date_range('2015-01', periods=120, freq='ME'))

# ADF 검정
result = adfuller(ts)
print(f"p-value: {result[1]:.4f}")  # 0.05 이상 → 비정상

# 1차 차분
ts_diff = ts.diff().dropna()
result2 = adfuller(ts_diff)
print(f"차분 후 p-value: {result2[1]:.4f}")  # 0.05 미만 → 정상
```

**Step 2. ACF/PACF로 p, q 결정**

```python
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
plot_acf(ts_diff, lags=20, ax=axes[0])
plot_pacf(ts_diff, lags=20, ax=axes[1])
plt.tight_layout()
plt.show()
```

**Step 3. ARIMA 모델 학습 및 예측**

```python
from statsmodels.tsa.arima.model import ARIMA

# ARIMA(1, 1, 1) — 차분 1회, AR 1차, MA 1차
model = ARIMA(ts, order=(1, 1, 1))
result = model.fit()
print(result.summary())

# 12개월 예측
forecast = result.forecast(steps=12)
print(forecast)
```

**Step 4. 자동으로 최적 파라미터 찾기 (auto_arima)**

```python
# pip install pmdarima
from pmdarima import auto_arima

model = auto_arima(ts, seasonal=False, stepwise=True, information_criterion='aic')
print(f"최적 order: {model.order}")
```

## 5. ARIMA의 한계

| 한계 | 설명 |
|---|---|
| 선형 관계만 모델링 | 비선형 패턴 포착 불가 |
| 단변량 | 외부 변수 반영 어려움 |
| 장기 의존성 약함 | 먼 과거의 영향 반영 한계 |
| 수동 파라미터 설정 | p, d, q 결정에 경험 필요 |

이 한계를 극복하기 위해 등장한 것이 RNN, LSTM 같은 딥러닝 모델이다.

---

## 정리

- ARIMA(p, d, q) = AR + 차분 + MA
- 비정상 시계열은 차분(d)으로 정상화
- ACF → q, PACF → p 결정
- auto_arima로 최적 파라미터 자동 탐색 가능
- 선형 모델의 한계 → 딥러닝으로 이어짐

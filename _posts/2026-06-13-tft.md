## 들어가며

[지난 글](../lstm-multivariate)에서 다변량 LSTM의 한계를 확인했다.
변수 간 중요도를 반영하지 못하고, 장기 의존성도 완전히 해결되지 않았다.
**TFT(Temporal Fusion Transformer)**는 Attention 메커니즘을 활용해
어떤 변수가, 어떤 시점에 얼마나 중요한지를 학습한다.
Google이 2021년 발표한 모델로, 시계열 예측 분야에서 큰 주목을 받았다.

---

## 1. TFT가 해결하는 문제

| 문제 | LSTM | TFT |
|---|---|---|
| 장기 의존성 | 약함 | Self-Attention으로 해결 |
| 변수 중요도 | 모두 동일 | Variable Selection Network |
| 정적/동적 변수 구분 | 불가 | 명시적 구분 |
| 예측 불확실성 | 단일 값 | 분위수(Quantile) 예측 |
| 해석 가능성 | 낮음 | Attention 가중치로 해석 |

---

## 2. TFT 핵심 구조

### Variable Selection Network (VSN)

입력 변수마다 중요도를 학습해 가중치를 부여한다.
어떤 변수가 예측에 얼마나 기여했는지 직접 확인할 수 있다.

### Gated Residual Network (GRN)

불필요한 정보를 게이트로 걸러내고 잔차 연결로 학습을 안정화한다.

$$GRN(x) = LayerNorm(x + GLU(\eta_1))$$
$$\eta_1 = W_1 \cdot ELU(W_2 x + W_3 c + b_2) + b_1$$

- $c$ : 선택적 컨텍스트 벡터
- GLU : Gated Linear Unit

### Interpretable Multi-Head Attention

Transformer의 Multi-Head Attention을 변형해
**시간적 패턴에 대한 해석 가능한 Attention**을 제공한다.

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

각 시점의 Attention 가중치를 시각화하면
모델이 어떤 과거 시점을 참고해서 예측했는지 확인할 수 있다.

### Quantile Prediction

단일 예측값 대신 **분위수(Quantile)**로 예측 범위를 제공한다.

- $q_{0.1}$ : 하한 (10%)
- $q_{0.5}$ : 중간값 (50%, 실제 예측)
- $q_{0.9}$ : 상한 (90%)

이를 통해 예측의 불확실성을 정량화할 수 있다.

---

## 3. 입력 변수 종류

TFT는 변수를 세 가지로 구분해서 처리한다:

| 종류 | 설명 | 예시 |
|---|---|---|
| Static Covariates | 시간에 따라 변하지 않는 변수 | 지역, 카테고리 |
| Known Future Inputs | 미래를 미리 알 수 있는 변수 | 요일, 공휴일, 프로모션 |
| Observed Inputs | 과거만 알 수 있는 변수 | 과거 매출, 기온 |

LSTM은 이 구분 없이 모든 변수를 동일하게 처리했지만,
TFT는 각 변수의 특성에 맞게 다르게 인코딩한다.

## 4. PyTorch Forecasting으로 구현

```python
# pip install pytorch-forecasting pytorch-lightning

import pandas as pd
import numpy as np
import torch
from pytorch_forecasting import TemporalFusionTransformer, TimeSeriesDataSet
from pytorch_forecasting.metrics import QuantileLoss
import matplotlib.font_manager as fm
import matplotlib.pyplot as plt

font_candidates = [
    'Noto Sans CJK JP', 'Noto Sans CJK KR',
    'NanumGothic', 'Malgun Gothic', 'AppleGothic', 'DejaVu Sans'
]
available_fonts = {f.name for f in fm.fontManager.ttflist}
selected_font = next((f for f in font_candidates if f in available_fonts), 'DejaVu Sans')
plt.rcParams['font.family'] = selected_font
plt.rcParams['axes.unicode_minus'] = False

# 예시 데이터 생성
np.random.seed(42)
n = 365
df = pd.DataFrame({
    'time_idx': range(n),
    'value': np.sin(np.linspace(0, 8*np.pi, n)) + np.random.normal(0, 0.1, n),
    'group': 'series_1',
    'month': pd.date_range('2023-01-01', periods=n).month.astype(str)
})

# TimeSeriesDataSet 생성
max_encoder_length = 30
max_prediction_length = 7

training = TimeSeriesDataSet(
    df[df.time_idx <= n - max_prediction_length - 1],
    time_idx='time_idx',
    target='value',
    group_ids=['group'],
    max_encoder_length=max_encoder_length,
    max_prediction_length=max_prediction_length,
    time_varying_known_categoricals=['month'],
    time_varying_unknown_reals=['value'],
)

validation = TimeSeriesDataSet.from_dataset(
    training, df, predict=True, stop_randomization=True
)

train_dataloader = training.to_dataloader(train=True, batch_size=32)
val_dataloader = validation.to_dataloader(train=False, batch_size=32)

# TFT 모델
tft = TemporalFusionTransformer.from_dataset(
    training,
    learning_rate=0.03,
    hidden_size=16,
    attention_head_size=1,
    dropout=0.1,
    hidden_continuous_size=8,
    loss=QuantileLoss(),
)

print(f"파라미터 수: {tft.size()/1e3:.1f}k")
```

---

## 5. Attention 시각화

TFT의 가장 큰 장점은 **예측 결과를 해석**할 수 있다는 점이다.

```python
# 학습 후 예측 및 시각화
predictions = tft.predict(val_dataloader, return_x=True)

# Variable Importance 확인
interpretation = tft.interpret_output(predictions.output, reduction='sum')
tft.plot_interpretation(interpretation)
plt.show()
```

Attention 가중치 시각화를 통해:
- 어떤 변수가 중요한지
- 어떤 과거 시점을 참고했는지
- 예측 불확실성 범위는 어느 정도인지

를 직접 확인할 수 있다.

---

## 정리

- TFT는 Variable Selection + GRN + Attention + Quantile 예측을 결합
- 변수를 Static / Known Future / Observed로 구분해 처리
- Attention으로 어떤 시점/변수가 중요한지 해석 가능
- Quantile 예측으로 불확실성 범위 제공
- LSTM 대비 해석 가능성과 장기 의존성 모두 향상
